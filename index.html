How to Play

Sudoku is a logic-based number puzzle. The goal is to fill a 9×9 grid so that every row, column, and 3×3 box contains the digits 1 through 9 exactly once.

1
Each row must contain 1–9 with no repeats.
2
Each column must contain 1–9 with no repeats.
3
Each 3×3 box must contain 1–9 with no repeats.
